Admin Miscellaneous

Overview

Admin Miscellaneous page is used for simple miscellaneous administration operations This page includes an on demand refresh of application cache (only one for now).

It's broken up into Acceptance Tests that are Automated, Manual Tests and items that are known but deemed Out of Scope.

The Notes and Resources section contains notes and links to various documents/resources related to this page/feature.

Acceptance Tests (Automated)

  • When clicking on the Refresh Cache button the cache will be refreshed and a "Refresh cache process completed successfully" message will occur.

Manual Tests

  • Clicking on "Refresh Cache" stays on the current page and displays a success message when the refresh operation is complete
  • Check User Interface is consistent with the Usability Guidelines.

Untested (Out of Scope)

  • Performance/Load testing of "Refresh Cache" is out of scope. If it does not impact the users during QA the assumption is this will not change in production.
